# 약물·카페인 농도·약효 그래프

교육용 정적 웹앱입니다. 약물과 카페인의 복용량, 복용 시각, 피크 시간, 반감기, 그래프 표시 시간을 직접 입력해 상대 농도 변화를 봅니다. 기본 그래프 범위는 12시간입니다.

그래프는 혈중 농도 추정선과 약효/각성 구간을 분리해서 보여줍니다. 콘서타와 카페인은 당일 지속 구간을 표시하고, 아빌리파이와 에스시탐(에스시탈로프람 가정)은 긴 반감기와 정상상태 도달 시간을 별도로 표시합니다.

각 항목의 `매일 반복` 체크를 켜면 같은 시각에 매일 복용한 것으로 보고, 과거/미래 반복 복용분을 그래프에 합산합니다.

## 근거

- Concerta: DailyMed 약동학 수치와 OROS methylphenidate 12.5시간 효과 연구
- Abilify: DailyMed 약동학 수치, 긴 반감기와 약 14일 정상상태
- Escitam: escitalopram으로 가정, DailyMed 약동학 수치와 SSRI 임상 효과 문헌
- Caffeine: NCBI StatPearls 및 공개 리뷰의 성인 평균 반감기/피크 범위

## 사용

`index.html`을 브라우저에서 열면 됩니다. 별도 빌드가 필요 없습니다.

## 주의

이 앱은 의료 조언이나 복용 지시가 아닙니다. 실제 혈중 농도, 효과, 부작용을 보장하지 않습니다. 복용량이나 복용 시간 변경은 처방 의료진과 상의해야 합니다.
